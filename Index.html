💡 What do these roles mean?
Target Variable (required): The number you want to predict. E.g., stock price, revenue, points scored.

Date / Time (optional): When each data point happened. Enables time-series forecasting. E.g., 2026-01-01.

Label / Name (optional): A text identifier for each row. E.g., team name, product SKU.

Feature (optional): Extra numbers that help predict the target. E.g., volume traded, units sold, temperature.

No role: Column is ignored in the prediction.
